🔹 الجزء الأول: حل المعادلات المثلثية

  1. 1. عند حل المعادلة sin θ cos θ - cos θ = 0 بالتحليل، نأخذ العامل المشترك:
    ✅ الإجابة: ب) نخرج cos θ كعامل مشترك لتصبح cos θ(sin θ - 1/2) = 0
  2. 2. لتوحيد الدوال في معادلة تحتوي على sin²θ و cos θ، نعوض بـ:
    ✅ الإجابة: ب) من متطابقة فيثاغورس: sin²θ + cos²θ = 1 → sin²θ = 1 - cos²θ
  3. 3. إذا نتج عن الحل قيمة cos θ = 3/2، فإن هذا الحل:
    ✅ الإجابة: ب) قيم cos محصورة بين -1 و 1، لذا 3/2 مرفوض
  4. 4. عند تربيع طرفي معادلة مثلثية، يجب التحقق من الحلول لأن:
    ✅ الإجابة: ب) التربيع قد يُنتج حلولاً إضافية يجب استبعادها بالتعويض

🔹 الجزء الثاني: التطبيقات الفيزيائية (الحركة الدورية)

  1. 5. في معادلة الحركة الدورية h = 21 - 20 cos(3πt)، يمثل الرقم 21:
    ✅ الإجابة: ب) الثابت المضاف يمثل خط الاتزان أو المتوسط
  2. 6. لإيجاد الزمن t عندما يكون h = 1 في المعادلة السابقة، نحل المعادلة:
    ✅ الإجابة: أ) نعوض h = 1 في المعادلة الأصلية
  3. 7. بعد تبسيط المعادلة 1 = 21 - 20 cos(3πt)، نحصل على:
    ✅ الإجابة: ج) -20 = -20 cos(3πt) → cos(3πt) = 1
  4. 8. أقل زمن موجب t يحقق cos(3πt) = 1 هو:
    ✅ الإجابة: ب) 3πt = 2kπ → t = 2k/3، لأقل زمن موجب k=1 → t=2/3

🔹 الجزء الثالث: SAT/ACT - الدوال العكسية والمتتاليات

  1. 9. قيمة sin(sin⁻¹(x)) تساوي:
    ✅ الإجابة: ب) الدالة وعكسها تلغيان بعضهما ضمن المجال
  2. 10. لحساب cos(sin⁻¹(x))، نستخدم مثلثاً قائماً حيث المقابل = x والوتر = 1، فيكون المجاور:
    ✅ الإجابة: ج) حسب فيثاغورس: المجاور = √(الوتر² - المقابل²)
  3. 11. صيغة مجموع المتتالية الحسابية Sₙ هي:
    ✅ الإجابة: أ) Sₙ = n/2 (a₁ + aₙ)
  4. 12. إذا كان a₁ = -13, aₙ = 427, Sₙ = 18423، فإن عدد الحدود n يساوي:
    ✅ الإجابة: ب) 18423 = n/2(-13+427) → 18423 = 207n → n = 89

  1. 13. وفقاً لمتطابقات الزوايا المتممة، cos(90° - θ) تساوي:
    ✅ الإجابة: ب) cos(90° - θ) = sin θ
  2. 14. قيمة sin(270° - θ) تساوي:

    ✅ الإجابة: ب) (-2)⁴ + 10(-2)² + (-2) + 8 = 16 + 40 - 2 + 8 = 62
  4. 16. النصيحة الذهبية عند رؤية tan⁴θ و sec²θ في نفس المعادلة هي:
